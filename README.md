# ZAYN RUSH — Online Racing Game

Multiplayer racing game built in Python + Pygame with a hybrid client–server + peer-to-peer architecture (TCP/UDP). Includes real-time synchronization (join/leave, position updates, collisions), multithreaded networking, and an SQLite persistence layer for player profiles and match history.

## Features
- Multiplayer networking (TCP/UDP)
- Real-time sync + collision events
- SQLite database for profiles and history
- Full game UI + assets (maps, cars, obstacles, sounds)

## Tech Stack
Python • Pygame • PyQt5 • TCP/UDP • SQLite • Multithreading
